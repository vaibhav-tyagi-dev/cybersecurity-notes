 ## Linux Basic commands  
  
  - ####Note : Linux commands are case sensitive.
  
  - `pwd` - Print working directory (Shows user's current location)  
  - `whoami` - Shows logged in user  
  - `uname -a` - System information  
  - `ls` - List files/directories  
  - `ls -l` - List files in long format (permission, owner, size, last modified date)  
  - `ls -a` - List hidden files  
  - `ls -la` - Long format and hidden files  
  - `mkdir dir` - Create directoy  
  - `cd` - Change and enter directory  
  - `cd ..` - Go to parent directory  
  - `cd ~` - Home directory  
  - `touch` - Create empty file  
  - `nano` - Edit file (Ctrl + o - Enter to save, Ctrl + x to exit)  
  - `cat` - Read file  
  - `cp` - Make copy of file  
  - `mv` - Move/Rename file  
  - `rm` - Delete file  
  - `rm -r` - Delete directory  
  - `chmod` - Change permission  
  - `man ls` - Manual/Documentation  
  - `--help` - Help
