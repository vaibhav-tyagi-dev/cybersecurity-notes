 # Linux commands  
  
   #### Note : Linux commands are case sensitive.  
   
  - `echo` - Displays the text or arguments  
  - `pwd` - Print working directory (Shows user's current dirextory)  
  - `whoami` - Shows logged in user  
  - `uname -a` - System information  
  - `ls` - List files/directories  
  - `ls -l` - List files in long format (permission, owner, size, last modified date)  
  - `ls -a` - List hidden files  
  - `ls -r` - List files/dir in reverse order  
  - `ls -la` - Long format and hidden files  
  - `mkdir dir` - Create directoy  
  - `cd` - Change and enter directory  
  - `cd .` - Current directory  
  - `cd ..` - Go to parent directory  
  - `cd ~` - Home directory  
  - `cd -` - Previous directory  
  - `touch` - Create empty file (If the file already exists the timestamp will change)  
  - `touch -r` - Using reference file (Match the timestamp of multiple files)  
  - `touch -d` - Setting a specific date of file  
  - `file` - Find format of the file  
  - `nano` - Edit file (Ctrl + o - Enter to save, Ctrl + x to exit)  
  - `cat` - View short/single page text file  
  - `cat file1 file2` - Display combined output
  - `cat >` - Write text into a file directly (Write - Ctrl + D to save)new file can be created also including text.  
  - `cat -n` - Numbers all output lines  
  - `cat -b` - Numbers non-empty output lines
  - `less` - View large text files[(less - /search_term - enter ) To search a word]  
  - `cp` - Make copy of file  
  - `mv` - Move/Rename file  
  - `rm` - Delete file  
  - `rm -r` - Delete directory  
  - `chmod` - Change permission  
  - `man ls` - Manual/Documentation  
  - `--help` - Help
